# Security Scripts

A collection of custom scripts built during my offensive security learning journey. 
Each tool here started as a concept I wanted to understand deeply before using 
automated alternatives — the goal is always to know what a tool does before letting 
it do it for you.

## Scripts

**Python TCP Port Scanner**  
A lightweight TCP port scanner built from scratch using Python's socket library. 
Tested against Metasploitable 2 as part of home lab practice.  
[View Project](port-scanner/)
